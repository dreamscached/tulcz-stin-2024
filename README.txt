Zde je úložiště zdrojového kódu pro semestrální práci STIN 2024, kterou implementoval German Semin.
Nasazení využívá Ansible & Github Actions pro nahrávání Docker image a restartování kontejnerů. Server je hostován na Hetzneru.

Pull Requests: https://github.com/dreamscached/tulcz-stin-2024/pulls
Workflow runs: https://github.com/dreamscached/tulcz-stin-2024/actions
Server nasazen na: https://weather.nullhaven.one/
Dokumentace API: https://weather.nullhaven.one/docs

DSP dostupné na: https://github.com/dreamscached/tulcz-stin-2024/blob/master/docs/DSP.txt
Používa se Cloudflare za obranu proti DDoS a exploitu.
